## Install

```bash
go install github.com/vikbert/random
```

```bash
random help
```

## Usage

```bash
random -h
random uuid -h
random pass -h
```

![](assets/help.png)
![](assets/usage.png)


| Commands  | Description |
|---|---|
| uuid  | generate & paste an UUID ✅ |
| pass | generate & paste a password ✅ |
| text  |  generate & paste a dummy text ✅ |
| joke  |  display a random joke ✅ |
| motto  |  display a random motto ✅ |
