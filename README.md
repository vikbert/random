## Install
```bash
go install github.com/vikbert/random
```
```bash
random -h
```

## Usage
```bash
$ random -h
$ random uuid
$ random uuid -h
$ random pass -h
```

## Development
```bash
git clone https://github.com/vikbert/random.git
cd random
go run main.go
```
